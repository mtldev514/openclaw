# OpenClaw

Personal AI assistant platform. Multi-platform CLI and gateway that connects to messaging channels (WhatsApp, Telegram, Discord, Slack, Signal, iMessage, Matrix, Zalo) with multiple AI model backends (Anthropic Claude, OpenAI).

Monorepo with native apps (iOS, Android, macOS), plugin/extension system, voice calls, web gateway, RPC agent mode, and TUI/web interfaces.

Node.js 22+, TypeScript, pnpm, Vitest.
